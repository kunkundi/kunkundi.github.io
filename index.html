核心功能 Core Features

🖥️ 跨平台支持 Cross-platform Support

全面兼容 Windows、macOS 与 Linux 操作系统，满足多平台远程访问需求。 Fully compatible with Windows, macOS, and Linux to meet multi-platform remote needs.

⚡ 硬件加速支持 Hardware Acceleration

支持 NVIDIA Video Codec、VideoToolbox 硬件加速，支持 H.264、AV1 编码格式。 Supports NVIDIA Video Codec, VideoToolbox acceleration, and H.264/AV1 formats.

∞ 不限设备数量 Unlimited Devices

支持同时管理和连接无限数量的远程设备，满足多场景需求。 Manage and connect unlimited remote devices for all use cases.

安装与使用 Installation 

git clone https://github.com/kunkundi/deskport.git
cd DeskPort
git submodule update --init
xmake b remote_desk
xmake r remote_desk
