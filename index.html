核心功能

🖥️ 跨平台支持

全面兼容 Windows、macOS 与 Linux 操作系统，满足多平台远程访问需求。

⚡ 硬件加速支持

支持 NVIDIA Video Codec、VideoToolbox 硬件加速，支持 H.264、AV1 编码格式。

∞ 不限设备数量

支持同时管理和连接无限数量的远程设备，满足多场景需求。

安装与使用

git clone https://github.com/dijunkun/DeskPort.git
cd DeskPort
git submodule update --init
xmake b remote_desk
xmake r remote_desk

更多用法请见 README.md
